# ✅ Todo App using React

## 📌 Overview
A simple **Todo App** built using **React & Hooks** that allows users to **add, view, and delete** tasks dynamically. The app features a clean UI with responsive components.

## 🚀 Features
- Add, view, and delete todo items 📝  
- Dynamic list rendering using **React state** 🔄  
- Displays a **welcome message** when no tasks are present 🎉  
- **Component-based architecture** for easy maintenance 🏗️  
- Styled using **CSS Modules** for scoped styling 🎨  

## 🛠️ Tech Stack
- **React.js** ⚛️
- **CSS Modules** 🎨
- **Bootstrap (optional for styling)** 💅

## 📦 Installation & Setup
1. **Clone the repository**
   ```bash
   git clone https://github.com/Kailramiya/Todo/
   cd your-repo-folder
2. **Install dependencies**
   ```bash
    npm install
   ```
3. **Run the development server**
   ```bash
    npm run dev
4. **Open in Browser**
    Navigate to http://localhost:3000 to view the app.
